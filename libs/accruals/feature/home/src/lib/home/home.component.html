Accruals Application
This application is an easy to use tool to help manage savings to meet bills and other financial commitments that will arise at some future date, for example your annual Building & Contents home insurance.
Technologies
The application was developed as a demonstration application using the Angular framework to showcase a number of different technologies. Angular Material provides a fully featured set of UI components and Tailwind CSS is used for styling.
Firebase provides the backend services, using Firebase Authentication for an Email/Password based authentication service and Firestore for the database service.
The application uses standalone components throughout (module-less).
The build system used is Nx by Nrwl.
